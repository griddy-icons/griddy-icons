## Usage

```jsx
import { Activity } from "griddy-icons";

const App = () => {
  return (
    <div>
      <Activity size={32} filled />
    </div>
  );
};
```

## Configuration

Icons accept standard SVG props plus:

- `size`: width and height (default: "24px")
- `filled`: boolean (default: false)
- `color`: stroke/fill color (default: "currentColor")
